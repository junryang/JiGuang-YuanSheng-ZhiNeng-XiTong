### 一、项目管理中心概述

项目管理中心是**项目智能体团队**的核心工作界面，由**项目部**的智能体团队负责。主脑（CEO）管理项目团队，项目经理统筹具体项目执行。项目团队包括：项目经理、项目协调员、项目助理等智能体。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部项目，审批重大决策，查看项目健康度 |
| CEO/主脑 | 管理项目团队，审批立项，协调资源 |
| 领域总经理 | 管理领域内项目组合，审批项目计划 |
| 项目经理 | 管理具体项目，分配任务，跟踪进度 |
| 项目协调员 | 协调跨部门协作，解决依赖冲突 |
| 项目助理 | 辅助项目管理，整理文档，跟踪任务 |
| 部门主管 | 查看参与项目，管理本部门任务 |
| 员工 | 查看分配任务，更新任务进度 |

### 七、项目动态处理能力

| 动态问题类型 | 检测方式 | 处理方式 | 负责智能体 | 升级条件 |
|-------------|---------|---------|-----------|---------|
| 进度滞后 | 实际vs计划对比 | 自动告警，建议调整资源或计划 | 项目经理 | 滞后超过15% |
| 预算超支 | 实时预算跟踪 | 自动告警，建议控制支出 | 财务主管 | 超支超过10% |
| 资源冲突 | 资源分配检测 | 自动识别冲突，建议协调 | 项目协调员 | 影响关键路径 |
| 任务阻塞 | 任务状态监控 | 自动识别阻塞，通知负责人 | 项目助理 | 阻塞超过24小时 |
| 依赖延期 | 依赖链分析 | 自动预警，调整下游计划 | 项目协调员 | 影响里程碑 |
| 负载不均 | 负载均衡检测 | 自动建议重新分配任务 | 项目经理 | 单人体负载超过80% |
| 能力缺口 | 任务需求vs能力库 | 自动识别，建议培训或外包 | 智能体主管 | 影响项目交付 |
| 风险触发 | 风险监控 | 自动执行应对措施，通知相关人 | 项目经理 | 风险等级升高 |
| 里程碑延期 | 里程碑跟踪 | 自动预警，建议调整计划 | 项目经理 | 延期超过3天 |
| 团队协作问题 | 沟通频率分析 | 自动识别，建议协调会议 | 项目协调员 | 持续3天无进展 |

## 项目智能体团队职责分工

| 智能体 | 层级 | 岗位 | 核心职责 |
|--------|------|------|---------|
| 项目经理 | L3 | 项目负责人 | 统筹项目执行，管理进度、预算、风险，向主脑汇报 |
| 项目协调员 | L4 | 协调专员 | 协调跨部门协作，解决依赖冲突，促进沟通 |
| 项目助理 | L5 | 助理 | 跟踪任务进度，整理文档，编制周报 |
| 部门主管 | L4 | 部门负责人 | 管理本部门任务，分配资源，质量把控 |
| 员工 | L5 | 执行者 | 执行具体任务，更新进度，报告问题 |


## 与主脑的协作机制

| 事项类型 | 处理方式 | 升级条件 |
|---------|---------|---------|
| 日常任务分配 | 项目经理自主处理 | 无需升级 |
| 项目进度跟踪 | 项目助理定期更新 | 无需升级 |
| 进度滞后 | 项目经理调整计划 | 滞后超过15%需向主脑汇报 |
| 资源不足 | 项目经理申请资源 | 需主脑审批 |
| 项目立项 | 项目经理提交计划书 | 需主脑审批 |
| 项目变更 | 项目经理提交变更申请 | 影响范围大时需主脑审批 |
| 风险预警 | 项目经理评估后处理 | 重大风险需向主脑汇报 |
| 项目复盘 | 项目经理组织复盘 | 复盘报告需主脑审阅 |
