# 纪光元生智能系统 - 界面设计文档 v1.0

## 客户服务中心（新增）


### 一、客户服务中心概述

客户服务中心是**客户成功部**的核心工作界面，由**主脑（CEO）直接管辖**，负责管理所有项目客户、外部平台客户、接单平台客户、媒体发稿客户等全部客户关系。客户成功部智能体团队负责日常客户服务工作。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部客户数据，审批重大客户决策 |
| 主脑/CEO | 直接管辖客户服务中心，管理客户成功团队 |
| 客户成功主管 | 管理全部客户服务功能，分配客户服务任务 |
| 客户成功经理 | 管理指定客户群体，处理客户需求 |
| 技术支持工程师 | 提供技术支持和问题解决 |
| 客户培训师 | 培训客户使用系统，制作培训材料 |
| 实习客户助理 | 辅助客户服务，整理客户资料 |

## 十、客户成功部团队职责分工

| 智能体 | 层级 | 岗位 | 核心职责 |
|--------|------|------|---------|
| 客户成功主管 | L4 | 客户成功负责人 | 统筹客户服务工作，审核服务质量，管理团队，向主脑汇报 |
| 客户成功经理A | L5 | 客户成功经理 | 管理项目客户关系，跟进客户需求，处理客户问题，推动续约 |
| 客户成功经理B | L5 | 客户成功经理 | 管理平台客户和接单客户，维护粉丝关系，提升客户满意度 |
| 技术支持工程师A | L5 | 技术支持 | 处理技术工单，解决客户技术问题，提供部署支持 |
| 技术支持工程师B | L5 | 技术支持 | 处理技术工单，编写技术文档，支持客户系统集成 |
| 客户培训师 | L5 | 客户培训 | 培训客户使用系统，制作培训材料，提升客户使用深度 |
| 实习客户助理 | L6 | 助理 | 整理客户资料，辅助工单分类，协助客户服务 |


## 十一、与主脑的协作机制

| 事项类型 | 处理方式 | 升级条件 |
|---------|---------|---------|
| 日常客户服务 | 客户成功团队自主处理 | 无需升级 |
| 紧急工单 | 技术支持工程师立即处理 | 影响客户业务时向主脑汇报 |
| 客户投诉 | 客户成功主管评估后处理 | 严重投诉向主脑汇报 |
| 客户流失风险 | 客户成功经理主动联系 | 高价值客户流失风险向主脑汇报 |
| 续约谈判 | 客户成功经理跟进 | 涉及折扣或合同变更需主脑审批 |
| 团队扩编 | 客户成功主管申请 | 需主脑审批 |
| 重大客户决策 | 客户成功主管提案 | 需主脑审批 |
