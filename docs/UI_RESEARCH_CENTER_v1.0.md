### 一、市场调研中心概述

市场调研中心是**战略决策层**的核心工具，帮助老板和管理层全面掌握AI可拓展领域、互联网创业赛道、传统行业智能化改造等全领域的市场机会。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部领域，进行战略决策 |
| 合伙人 | 查看全部领域，参与战略决策 |
| CEO | 查看全部领域，制定领域规划 |
| 战略分析师 | 查看全部领域，撰写分析报告 |
| 行业研究员 | 查看指定领域，收集整理信息 |
