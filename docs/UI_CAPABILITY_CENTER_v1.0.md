## 智能体能力库与工具库中心（新增）


### 一、能力库中心概述

能力库中心是**智能体能力的核心管理界面**，展示所有智能体已具备的能力、开发中的能力、待开发的能力，并支持一键通知主脑处理能力缺口问题。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部能力，发起能力建设 |
| CEO/主脑 | 管理全部能力，分配能力开发任务 |
| CTO | 查看全部能力，评估技术可行性 |
| 智能体主管 | 管理智能体能力配置 |
| 智能体工程师 | 查看能力详情，执行能力开发 |
