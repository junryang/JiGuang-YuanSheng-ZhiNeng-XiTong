## 一、引言

### 1.1 关于本手册

本手册面向**纪光元生智能系统（JYIS）**的最终用户，帮助您快速上手使用系统的各项功能。手册中的所有功能均基于 `AGENT_ABILITY_SPEC_v1.0.md` 中定义的通用能力实现，您可以在使用过程中体会到智能体真正的“智能”之处。

### 1.2 系统简介

纪光元生智能系统是一个**人机协同的多智能体开发平台**。您作为决策者（老板），只需要与CEO智能体（主脑）对话，系统会自动完成任务的分解、分配和执行。

**核心特点**：
- 🤖 **人机协同**：您只对接CEO智能体，无需关心具体执行细节
- 🧠 **组织化管理**：智能体按照企业组织架构分级管理
- 📋 **全生命周期**：从市场调研到项目复盘，全程自动化
- 🔌 **多模型支持**：可对接DeepSeek、GPT-4、Claude等多种大模型（对齐 EM-01 多模型路由）
- 💬 **可解释决策**：智能体能解释其决策逻辑，让您对过程了如指掌（对齐 AGENT-RUNTIME-03 决策可解释性）
- 📈 **持续进化**：系统能从反馈中学习，越用越聪明（对齐 LN-04 双循环学习）

### 1.3 用户角色说明

| 角色 | 说明 | 主要操作 | 关联能力 |
|------|------|---------|----------|
| 老板（BOSS） | 系统最高决策者 | 战略决策、审批立项、最终验收 | L0 层级 |
| CEO（主脑） | 智能体总负责人 | 理解意图、拆解目标、分配资源 | AGENT-RUNTIME-01 主循环 |
| 总经理 | 领域负责人 | 管理特定开发领域 | DC-01 任务规划 |
| 经理 | 项目负责人 | 管理具体项目 | DC-02 子任务分解 |
| 主管 | 部门负责人 | 管理部门工作、技术评审 | QL-05 质量验证 |
| 员工 | 任务执行者 | 执行具体任务 | EX-01 代码生成 |
| 实习 | 助理 | 辅助支持、信息检索 | MM-04 记忆检索 |

> **作为老板，您只需要关注第0层和第1层的操作。** 下层智能体会自动完成工作。

## 三、核心操作指南

### 3.1 与CEO智能体对话

**这是您最常用的功能。** 您只需要像跟真人CEO聊天一样，用自然语言说出您的需求。

**示例对话**：

```
你：我需要开发一个AI客服系统，支持微信公众号接入。

主脑：[思考中] 
正在分析需求类型... 识别为D01领域项目。
正在评估资源需求...

[回复]
收到。我已理解您的需求。

根据分析，这是一个D01网站开发领域项目，需要以下团队配合：
• 产品部：负责需求分析和产品设计
• 设计部：负责UI/UX设计
• 前端部：负责H5页面和公众号对接
• 后端部：负责API开发和AI模型接入
• 测试部：负责功能测试

预计项目周期：2周
预估预算：¥50,000

是否继续立项？

你：继续，立项。

主脑：[执行中]
正在创建项目计划书...
正在分配智能体团队...

[回复]
好的，已创建项目“AI客服系统 v1.0”。

项目负责人：Web总经理
预计开始时间：明天 09:00
预计完成时间：2026-02-03

我会定期向您汇报进度。
```

### 3.2 快捷指令

您可以使用以下快捷指令快速完成常见操作：

| 指令 | 功能 | 示例 | 关联能力 |
|------|------|------|----------|
| `创建项目` | 创建新项目 | "创建项目：开发一个智能体市场" | DC-01 任务规划 |
| `查看进度` | 查看当前项目进度 | "查看AI客服系统的进度" | CG-04 数值推理 |
| `分配任务` | 分配任务给智能体 | "把前端任务分配给资深前端工程师" | CL-01 任务委托 |
| `查看报告` | 查看项目报告 | "查看本周的项目报告" | EX-14 文档生成 |
| `审批` | 审批待办事项 | "批准智能体市场模块的开发预算" | APPROVE-02 审批流转 |
| `解释决策` | 让CEO解释决策理由 | "为什么选择这个方案？" | AGENT-RUNTIME-03 决策可解释性 |

### 3.3 查看项目进度

**方式一：对话查询**

```
你：查看所有项目的进度

主脑：当前进行中的项目：
1. AI客服系统 v1.0 - 进度 65% ████████░░░░
2. 智能体市场模块 - 进度 30% ███░░░░░░░░░

```

**方式二：点击项目卡片**

在底部状态栏点击项目名称，进入项目详情页查看详细信息。

### 3.4 审批操作

当有事项需要您审批时，系统会通过以下方式通知您：

1. **右上角通知图标**：显示红色数字表示待审批数量
2. **对话区域**：CEO智能体会主动提醒
3. **飞书/微信通知**：如果已连接，会发送消息通知（对齐 EX-08 消息发送）

**审批步骤**：

```
1. 点击右上角🔔图标
2. 查看待审批事项列表
3. 点击【批准】或【拒绝】
4. 可选填写审批意见
5. 确认提交
```

## 四、项目管理

### 4.1 创建项目

**方法一：对话创建**

```
你：创建一个新项目，开发一个具身智能机器人控制平台

主脑：好的，正在为您创建项目...
```

**方法二：手动创建**

1. 点击左侧菜单【项目列表】
2. 点击【创建项目】按钮
3. 填写项目信息（名称、领域、预算等）
4. 点击【提交】

### 4.2 查看项目详情

1. 在左侧组织架构树中，点击项目名称
2. 或通过对话：`查看[项目名称]的详情`

项目详情页包含：

| 模块 | 内容 |
|------|------|
| 基本信息 | 项目名称、编号、状态、进度 |
| 项目团队 | 参与该项目的所有智能体 |
| 任务列表 | 所有任务的名称、负责人、状态、进度 |
| 项目讨论 | 团队成员讨论记录 |

### 4.3 查看任务进度

在项目详情页的【任务列表】模块，可以看到：

- 任务名称
- 负责人（哪个智能体）
- 状态（进行中/已完成/待开始）
- 进度百分比
- 截止日期

### 4.4 项目复盘

项目完成后，系统会自动生成复盘报告（对齐 DC-02 子任务分解与总结）。

**查看方式**：
- 对话：`查看[项目名称]的复盘报告`
- 或在项目详情页点击【复盘报告】

**复盘报告包含**：
- 项目完成情况
- 实际耗时 vs 预估
- 实际成本 vs 预算
- 遇到的问题和解决方案
- 经验总结和建议

## 五、智能体管理

### 5.1 查看智能体

在左侧组织架构树中，点击任意智能体名称，可以查看：

- 基本信息（名称、职级、所属部门）
- 技能列表（该智能体具备的能力）
- 工作统计（完成任务数、代码行数等）
- **记忆系统**：该智能体记住的关键信息和经验（对齐 MM-03 长期记忆）

### 5.2 与智能体对话

您可以直接与任意层级的智能体对话：

1. 在组织架构树中点击智能体名称
2. 右侧会出现与该智能体的对话窗口
3. 输入您的指令

**示例**：

```
你（对资深后端工程师）：当前任务进展如何？

资深后端工程师：正在开发用户认证API，已完成60%，预计今天下午完成。
```

### 5.3 配置智能体技能

**权限要求**：主管及以上

1. 进入智能体详情页
2. 点击【技能库】模块的【编辑】
3. 勾选或取消勾选技能（对齐 META-01 能力扩展）
4. 点击【保存】

## 六、营销中心

### 6.1 进入营销中心

点击左侧菜单【营销中心】

### 6.2 内容管理

**创建内容**：

1. 点击【新建内容】
2. 选择平台（微信公众号/知乎/掘金/B站）
3. 输入标题和内容
4. 点击【保存草稿】或【发布】

**AI生成内容**（对齐 MK-01 文本内容生成）：

1. 在营销中心点击【AI生成】
2. 输入主题（如"多智能体系统架构"）
3. 选择平台和风格
4. 点击【生成】
5. 系统自动生成内容，您可编辑后发布

**多平台分发**（对齐 MK-08 多平台分发）：

1. 点击【批量发布】
2. 选择目标平台
3. 设置定时发布时间
4. 点击【确认发布】

### 6.3 数据分析

在营销中心查看：
- 粉丝增长趋势（对齐 CG-04 数值推理）
- 内容互动数据（点赞、评论、分享）
- 转化率统计
- **智能洞察**：系统自动发现的数据趋势和建议（对齐 CG-01 推理能力）

## 七、系统设置

### 7.1 个人设置

点击右上角头像 → 【个人设置】

可修改：
- 头像
- 密码
- 通知偏好

### 7.2 模型配置

点击左侧菜单【系统设置】→【模型配置】

可配置：
- 默认模型（DeepSeek/GPT-4/Claude）
- 备用模型（对齐 EM-03 模型降级）
- 模型参数（温度、最大Token等）

### 7.3 集成配置

点击左侧菜单【系统设置】→【集成配置】

可配置：
- 飞书集成（接收通知）
- 微信集成（接收通知）
- 企业微信集成

**连接飞书步骤**：

1. 点击【配置】按钮
2. 输入飞书机器人Webhook地址
3. 点击【测试连接】
4. 保存配置

## 八、常见问题

### 8.1 对话相关

**Q：CEO智能体不理解我的需求怎么办？**

A：请尝试：
1. 用更具体的语言描述需求
2. 分步骤描述，不要一次说太多
3. 使用快捷指令格式
4. 点击【展开思考过程】查看CEO的推理链路（对齐 AGENT-RUNTIME-03）

**Q：对话历史在哪里查看？**

A：点击左侧菜单【对话历史】，可以查看所有历史对话记录。

### 8.2 项目相关

**Q：如何修改项目信息？**

A：进入项目详情页，点击【编辑】按钮，修改后保存。注意：已立项的项目部分信息不可修改。

**Q：任务逾期了怎么办？**

A：系统会自动提醒任务负责人。您也可以：
1. 对话：`为什么[任务名称]逾期了？`
2. 手动重新分配任务给其他智能体

### 8.3 智能体相关

**Q：智能体状态显示"离线"怎么办？**

A：可能是网络问题或服务重启。请等待1-2分钟自动恢复，系统支持自愈机制（对齐 AGENT-RUNTIME-05 健康自检与自愈）。

**Q：如何创建新的智能体？**

A：目前智能体由系统自动创建和管理。如需自定义，请联系系统管理员。

### 8.4 账户相关

**Q：忘记密码怎么办？**

A：点击登录页的【忘记密码】，按照提示重置密码。

**Q：如何切换账号？**

A：点击右上角头像 → 【退出登录】，然后重新登录。
