### 一、财务中心概述

财务中心是**财务智能体团队**的核心工作界面，由**财务部**的智能体团队负责。主脑（CEO）管理财务团队，财务主管统筹日常工作。财务团队包括：财务主管、财务分析师、会计专员、出纳、税务专员等智能体。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部财务数据，审批重大财务决策 |
| CEO/主脑 | 管理财务团队，接收财务报告，审批预算 |
| CFO | 管理全部财务功能，审批支出，制定预算 |
| 合伙人 | 查看全部财务数据 |
| 财务主管 | 管理日常财务工作，审核报销，编制报表 |
| 财务分析师 | 分析财务数据，生成分析报告 |
| 会计专员 | 记账、对账、编制凭证 |
| 出纳 | 处理收付款、管理现金流 |
| 税务专员 | 税务申报、税务筹划 |

## 与主脑的协作机制

| 事项类型 | 处理方式 | 升级条件 |
|---------|---------|---------|
| 日常记账 | 会计专员自主处理 | 无需升级 |
| 预算内支出 | 出纳自主处理 | 无需升级 |
| 预算超支 | 财务主管审批后执行 | 超过10%需主脑审批 |
| 财务报表 | 财务主管审核后发布 | 重大异常需向主脑汇报 |
| 现金流预警 | 出纳通知财务主管 | 可用资金低于2个月支出时向主脑汇报 |
| 税务申报 | 税务专员自主处理 | 重大税务风险需向主脑汇报 |
| 对账差异 | 会计专员处理 | 差异超过¥5,000需财务主管审核 |
| 团队扩编 | 财务主管向主脑申请 | 人手不足时 |
