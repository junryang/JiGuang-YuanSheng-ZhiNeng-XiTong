# 文档模块索引 v1.0

## 一、用途

本索引用于快速定位项目设计文档，降低跨文档检索成本，并作为后续拆分治理入口。

## 二、架构与治理

- `PROJECT_OVERVIEW_v1.0.md`：项目总览与文档总入口  
- `PRD_v2.0.md`：产品范围与发布门槛基线  
- `ARCHITECTURE_v1.0.md`：系统架构设计  
- `ORGANIZATION_ARCHITECTURE_v1.0.md`：组织与层级架构  
- `LAYER_ABILITY_CONFIG_v1.0.md`：分层能力配置（含自上而下传导与主脑工具智能化 CEO-09～14）  
- `AGENT_ORG_SPEC_v1.0.md`：智能体组织规范  
- `DOCUMENTATION_BASELINE_v1.0.md`：跨文档统一基线  
- `PRODUCTION_RUNTIME_ASSUMPTIONS_v1.0.md`：生产运行假设与降级策略  
- `AGENT_MATURITY_MODEL_v1.0.md`：智能体成熟度分级与版本目标  
- `DESIGN_COVERAGE_AND_EXTENSIONS_v1.0.md`：设计覆盖分析（技能库/运行时工具/UI 缺口与能力扩展映射）  
- `DESIGN_CROSSCHECK_REPORT_v1.0.md`：设计文档交叉审查报告（营销/项目中心重点）
