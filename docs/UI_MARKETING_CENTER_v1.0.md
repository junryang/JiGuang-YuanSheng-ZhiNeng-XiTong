## 营销中心界面（新增）


### 一、营销中心概述

营销中心是**营销团队**的核心工作界面，负责内容创作、多平台分发、数据分析、竞品监控、接单变现、媒体发稿等全部营销相关功能。

| 使用角色 | 权限范围 |
|---------|---------|
| 老板 | 查看全部数据，审批关键内容 |
| 营销总经理 | 管理全部营销功能，审批内容 |
| 营销主管 | 管理内容、分发、数据分析 |
| 内容运营 | 创作内容、发布、查看数据 |
| 实习运营 | 辅助内容创作，只读数据 |
